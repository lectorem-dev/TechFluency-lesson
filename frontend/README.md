# Frontend учебной платформы

Frontend собран на React, TypeScript и Vite.

## Запуск в режиме разработки

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Что находится в `src`

- `pages/` — страницы приложения;
- `features/` — функциональные блоки по доменам;
- `shared/` — общие UI-компоненты, API-клиент и стили;
- `app/` — маршрутизация и корневое приложение.
