# Module 1: IT Jobs and Careers

## Vocabulary & Reading

The IT industry offers a variety of career paths. 

*   A **Software Developer** writes code for applications and maintains software systems. 
*   A **Data Scientist** analyzes large datasets to find patterns and insights that drive business decisions. 
*   A **DevOps Engineer** works on deployment automation and infrastructure management to ensure smooth software delivery. 
*   A **Cybersecurity Specialist** protects computer systems and networks from digital attacks and implements security measures. 
*   A **Network Administrator** manages an organization's computer networks, ensuring connectivity and performance.

### Matching

Match the job title with its description:

*   **DevOps Engineer**: Works on deployment automation and infrastructure management
*   **Network Administrator**: Manages an organization's computer networks
*   **Data Scientist**: Analyzes large datasets to find patterns and insights
*   **Cybersecurity Specialist**: Protects systems from digital attacks and implements security measures
*   **Software Developer**: Writes code for applications and maintains software systems

### True or False

1.  A Software Developer is primarily responsible for protecting systems from cyber attacks. **(False)**
    *   *Explanation: Software Developers write code for applications, while Cybersecurity Specialists handle protection from attacks.*
2.  DevOps Engineers focus on automating software deployment processes. **(True)**
    *   *Explanation: DevOps Engineers specialize in deployment automation and infrastructure management.*
3.  Network Administrators analyze large datasets for business insights. **(False)**
    *   *Explanation: Network Administrators manage computer networks, while Data Scientists handle data analysis.*
4.  Data Scientists help businesses make data-driven decisions. **(True)**
    *   *Explanation: Data Scientists analyze data to provide insights that inform business strategies.*
5.  Cybersecurity Specialists only work with antivirus software. **(False)**
    *   *Explanation: Cybersecurity Specialists implement various security measures including firewalls, encryption, and security protocols, not just antivirus software.*
